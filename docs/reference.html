Javascript Reference (for developers)


make_card(in_author, in_title, in_text, in_image, in_card_id, in_pages)

This is a function that generates new card news in the card news table. This only works on the client side. No requests are sent to the server.


Parameters

in_author

The author of the card news. This is a string of up to 17 characters.

in_title

The title of the card news. This is a string of up to 24 characters.

in_text

The content of the card news. This is a string of up to 64 characters.

in_image

The main image URL of the card news. This is a string of up to 2100 characters.

in_card_id

The unique id of card news. This is a unique integer.

in_pages

The number of pages in card news. This is an integer greater than or equal to zero.

Return value

This function does not return a value.

load_card_news(complete_function)

This is a function that receives a list of card news from the server and updates the contents of the card news table.


Parameters

complete_function[option]

This function is called after the card news list has been successfully received and the card news table has been updated.

Return value

This function does not return a value.

share(site, url)

This is a function that opens a window that shares a card news page with another SNS.


Parameters

site

The SNS to share card news page. ('facebook', 'twitter', 'google', ...)

url

The URL of the card news page to share.

Return value

This function does not return a value.

session_check()

This is a function that checks if the current session is valid. It gets the results from the server and changes the user's status to log out if the session is not valid.


Return value

This function does not return a value.

enable_loading()

This is a function that creates a loading mask on the page and blocks user input.


Return value

This function does not return a value.

disable_loading()

This is a function that removes the generated loading mask.


Return value

This function does not return a value.

enable_mask(name)

This is a function that creates a mask with a specific name on the page. Mask blocks user input and disappears when clicked.


Parameters

name

The name of the mask to create.

Return value

This function does not return a value.

disable_mask(name)

This is a function that removes the mask of the generated specific name.


Parameters

name

The name of the mask to remove.

Return value

This function does not return a value.

getQuerystring(paramName)

This is a function that extracts certain parameters from the query of the current url.


Parameters

paramName

The name of the parameter to extract.

Return value

The value of the extracted parameter. This is a string type.

open_card_news_viewer(id, edit_mode)

This is a function that opens specific card news to the card news viewer.


Parameters

id

The card is the ID of the target card news that is opened by the news viewer.

edit_mode[option]

Whether to enable edit mode. The default setting is false.

Return value

This function does not return a value.

set_state_login()

This function sets the current user's status to login. This only works on the client side. No requests are sent to the server.


Return value

This function does not return a value.

set_state_logout()

This function sets the current user's status to log out. This only works on the client side. No requests are sent to the server.


Return value

This function does not return a value.
