Setting up NewSTyle


Nginx and Python 3 are required to run NewSTyle.


1. Download the latest version of the program from the NewSTyle official page(https://stillnessfate.github.io/) or Github page(https://github.com/StillnessFate/stillnessfate.github.io).

2. Set the Nginx configuration file(nginx.conf) as shown below.

- Set root location to NewSTyle's html folder.

- Inside the server entry, add the following:

location ~ \.py$ {
    proxy_pass         http://127.0.0.1:8080;
    proxy_redirect     off;

    proxy_set_header   Host             $host;
    proxy_set_header   X-Real-IP        $remote_addr;
    proxy_set_header   X-Forwarded-For  $proxy_add_x_forwarded_for;
}

3. Run twisted_WSGI.py.

4. Run Nginx.


It's done! It is very simple.
