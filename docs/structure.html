Directory and File Structure


/twisted_WSGI.py

It is a WSGI server that connects web server (Nginx) and web application (/app.py).

It uses a twisted framework.


/app.py

It is a real web application.


/data/

Directory where various dynamic data are stored.


/data/database.db

An SQLite repository that stores user account information and card news information.


/data/empty.json

This is the default page of the editor. Used to edit the newly created card news page.


/data/cards/{cardnews_id}/{cardnews_pagenumber}.json

This is the space where the editor information of pages is saved. cardnews_id is a unique integer value, and cardnews_pagenumber is an integer value from 1 to more.

When card news is deleted, the cardnews_id directory will also be deleted.


/html/

The directory where web content is stored.


/html/css/

The directory where css files are stored.


/html/js/

The directory where javascript files are stored.


/html/grapesjs-dev/

The directory of the grapesjs framework. Used to edit the card news page.


/html/slick/

The directory of the slick library. Used to view card news pages.


/html/License.docx

This is the product's license warranty document.


/html/cards/{cardnews_id}/{cardnews_pagenumber}.html

This is the space where the actual card news pages are stored. cardnews_id is a unique integer value, and cardnews_pagenumber is an integer value from 1 to more.

When card news is deleted, the cardnews_id directory will also be deleted.
